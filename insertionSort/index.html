Insertion Sort

void insertionSort(int[] array) {
 int n = array.length;
 // Iterate through the array
 for (int i = 1; i < n; i++) {
  // Store the current element as the key
  int key = array[i];
  int j = i - 1;
  // Move elements that are greater than key one position ahead
  while (j >= 0 && array[j] > key) {
   array[j + 1] = array[j];
   j--;
  }
  // Place the key at the correct position
  array[j + 1] = key;
 }
}

More about Insertion Sort:

Insertion Sort is a simple and intuitive sorting algorithm used to sort an array by building the sorted portion of the array one element at a time. At each step, it inserts the current element into its correct position in the already sorted portion.

  • Key Features
    1. Simple and Easy to Implement: Insertion Sort is beginner-friendly and follows a step-by-step approach to sorting.
    2. In-Place Sorting: The algorithm does not require additional memory, making it space-efficient.
    3. Stable Sorting: It maintains the relative order of equal elements, making it suitable for applications where stability is required.
    4. Efficient for Small or Nearly Sorted Arrays: It performs well for small datasets or arrays that are already partially sorted.
  • Time Complexity
    • Best Case: O(n) - When the array is already sorted.
    • Average Case: O(n2).
    • Worst Case: O(n2) - When the array is sorted in reverse order.
  • Space Complexity: O(1)
  • Algorithm Steps
    1. Start with the Second Element: Assume the first element is already sorted.
    2. Select the Key Element: Take the current element as the key and compare it with elements in the sorted portion.
    3. Shift Elements: Move all elements greater than the key one position to the right.
    4. Insert the Key: Place the key in its correct position in the sorted portion.
    5. Repeat: Move to the next element and repeat the process until the entire array is sorted.
