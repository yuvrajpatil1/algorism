void bubbleSort(int[] array) {
 int n = array.length;
 for (int i = 0; i < n - 1; i++) {
  boolean swapped = false;
  for (int j = 0; j < n - i - 1; j++) {
   if (array[j] > array[j + 1]) {
    // Swap array[j] and array[j+1]
    int temp = array[j];
    array[j] = array[j + 1];
    array[j + 1] = temp;
    swapped = true; // Mark swapped as true
   }
  }
  if (!swapped) break;
 }
}

More about Bubble Sort:

Selection Sort is a straightforward and intuitive sorting algorithm used to sort an array by repeatedly selecting the smallest (or largest) element from the unsorted part and moving it to its correct position in the sorted part.

  • Key Features
    1. Simple and Easy to Implement: Selection Sort is a beginner-friendly algorithm with a straightforward approach to sorting.
    2. In-Place Sorting: The algorithm operates directly on the input array without requiring additional memory, making it space-efficient.
    3. Comparison-Based Algorithm: It compares elements to find the smallest (or largest) element during each iteration.

    1. Initialize the Unsorted Part: Start with the entire array as the unsorted portion.
    2. Find the Minimum Element: For each element in the unsorted portion, iterate through the remaining elements to find the smallest value.
    3. Swap the Minimum with the First Unsorted Element: Place the smallest element at the beginning of the unsorted part to expand the sorted portion.
    4. Repeat the Process: Move to the next position and repeat the process for the remaining unsorted portion of the array.
    5. Stop When All Elements Are Sorted: When the entire array is sorted, the algorithm terminates.
